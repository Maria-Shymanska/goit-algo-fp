# Висновки

## Task 7. Викоритстання методу Монте Карло

Симуляція методом Монте-Карло є ефективним інструментом для оцінки ймовірностей складних систем або подій. У нашому випадку, симуляція кидків кубиків дозволила отримати ймовірності сум значень, що випадають на кубиках, дуже близькі до теоретичних значень.
Використання великої кількості симуляцій (наприклад, 1 мільйон) забезпечує високу точність результатів, що дозволяє робити надійні висновки.
